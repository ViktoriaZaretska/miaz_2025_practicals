# Завдання 3 — Google Форма (Google Apps Script)

Інструкція:
1. Відкрий https://script.google.com/ і створи новий скрипт.
2. Скопіюй `google_form.gs` в редактор.
3. Натисни Run → authorize, обери акаунт і підтверди дозволи.
4. Запусти функцію `createForm`.
5. Відкрий View → Logs, щоб побачити URL створеної форми.

Форма містить питання: Ім'я, Дата, Відділ.
